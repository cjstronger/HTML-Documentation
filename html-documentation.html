Introduction

HTML is a front-end software language that allows one to build a website with elements. These elements are the main basis of the language, every new line starts with an element and ends with one. It is very basic on its own but when paired with other styling tools like CSS it can become an art.

The elements are very user-friendly, they make reading code very easy. The understandabilty of HTML makes it so reading the code allows the user to see what might end up where and the flow of the page. For example:

  • A body element is what holds all of your pages content and a section element is used as a separation piece to separate individual elements in the body.
  • HTML is also coded with a new element per line and the elements themselves are colored and indented in a way to understand where one ends and starts

HTML and CSS

HTML and CSS are like to peas in a pod, they were made for each other. HTML provides the code to allow CSS to manipulate those itemes into looking a different way.

HTML is the backbone of the web and is used to define the structure of web pages, including headings, paragraphs, lists, links, images, and other elements. Even websites built with more advanced technologies like JavaScript frameworks or content management systems (CMS) ultimately rely on HTML to render content in web browsers

Virtually all modern websites utilize CSS to some extent to control the visual presentation and layout of their content. CSS allows web developers to define styles such as colors, fonts, margins, padding, borders, and positioning, which are applied to HTML elements to create visually appealing and user-friendly web pages.

Hello World

To get started with writing HTML open your text-editor and input an opening <html> element. All elements start with a less than sign (opening tag) and end with a greater than sign (closing tag). Close the html element with the same setup used with a opening element but use a slash after the opening tag.

Once your code looks like the following:

<html>
</html>

Enter an opening and closing h1 element and in-between those, type the text 'Hello World'. Your code should look like the following:

          <html>
            <h1>Hello World</h1>
</html>

After you've typed your code, use the terminal tab in VSCode to start your code into your browser. I saved my file as 'test.html' so input for the terminal will be 'start test.html'. Hit enter, VSCode will ask your browser and Voila! See how easy that was?

The h's

In HTML, the <h1> to <h6> elements are used to define headings of different levels, where <h1> represents the highest level of heading and <h6> represents the lowest level of heading.

  • These elements represent headings of different levels, with <h1> being the highest level (most important) and <h6> being the lowest level (least important).
  • Headings form a hierarchical structure, where <h1> is typically used for the main heading or title of the page, and subsequent headings (<h2>, <h3>, etc.) are used for subheadings or sections within the page.
  • By default, browsers display headings with larger font sizes and more prominent styling to visually differentiate them from regular text. As you go down the ladder of the h's the font size decreases. However, the exact styling can be modified using CSS to match the design of the website.
